## 프로젝트 소개

- 프로젝트명: **YESRAE**
- 서비스 특징: 사용자 취향 기반 노래 추천 서비스
- 주요 기능
  - 노래 플레이리스트 추천
  - 유사도 기반 노래 퀴즈
  - 노래 이상형 월드컵
- 주요 기술
  - Annoy를 이용한 ANN 알고리즘
  - OAuth2.0, JWT
  - REST API
- 참조 리소스
  - JPA: 객체와 데이터베이스의 관계를 매핑
  - Querydsl: 쿼리 작성
  - Tailwind CSS: 디자인 전반 적용
  - Material Tailwind: 디자인 전반 적용
- 배포 환경
